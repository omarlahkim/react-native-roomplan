# react-native-roomplan

Bridge Integration for IOS RoomPlanKit

## Installation

```sh
npm install react-native-roomplan
```

## Usage

```js
import { RoomplanView } from "react-native-roomplan";

// ...

<RoomplanView color="tomato" />
```
